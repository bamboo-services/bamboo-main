# 介绍

筱主页「 XiaoMain 」是一个基于 Golang 开发的个人站点管理平台。它提供了一个简单的方式来管理你的个人站点，包括友链、我的项目、简历等。

> 起初目的就是为了方便我自己使用的一套系统，后面为了能够按照更好的标准化开发，索性还是配置为开源项目，方便大家使用。这样也会督促我自己写出规范的项目。
>
> —— 筱锋 xiao_lfeng
